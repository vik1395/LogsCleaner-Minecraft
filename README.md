This is a simple plugin that deletes all files inside the server logs folder every time the server starts. This is useful for large servers producing huge log files that just waste disk space. There is no config file, or commands.

Please report any issues with this plugin [HERE](https://github.com/vik1395/LogsCleaner-Minecraft/issues)
